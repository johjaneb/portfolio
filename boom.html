About the project

Boom is an individual project with the aim to make working remotely easier and more convenient. What is presented here is a prototype of a desktop application constructed in Adobe XD. The concept features two different modes, one for meetings as well as a creative mode for ideating, mindmapping and working on shared documents.

This project was constructed during a four week period in a course called Graphical interfaces. The focus of this project was to usability and to make a user interface that is clear and easy to use. The theory behind the project is from the book About face by Alan Cooper, Robert Reimann, David Cronin and Christopher Noessel.
